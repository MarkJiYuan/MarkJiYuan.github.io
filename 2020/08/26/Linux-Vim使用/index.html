Linux/Vim使用

Vim使用

  1. 使用/pattern进行向后搜索，?pattern向前搜索，回车确认，n为下一个结果，N为上一个结果

  2. h, j, k, l 分别是 左下上右

  3. 插入模式（Insert）

    Command Explaination
    i 在光标处插入
    o 向下插入一空行
    O 向上插入一空行
    a 在光标后插入
    A 在行尾插入

    Esc退出，回到Command模式

  4. 复制粘贴删除

    Command Explaination
    x 删除光标所在字符
    dd 删除一整行
    y 复制
    p 粘贴

  1. 保存与撤回

    Command Explaination
    :w 保存
    :q 退出
    :wq 保存并退出
    :q! 不保存就退出
    ZZ 相当于:wq，但更顺手
    u Ctrl + z
    Ctrl + r redo last action
  1. Vim强无敌！
