sqlite3使用说明

目前最新版本为5.0.0

  1. 安装

  2. Main

    1. new sqlite3.Database

      filename=”:memory:”只在内存中存储，程序结束释放。

      mode设置读写模式

      callback处理open和error event

    2. sqlite3.verbose()

      输出比较长的错误栈，用于开发时的调试。

  3. Database(假设instance是db)

    1. db.close

    2. db.configure

      设置sql执行回调等

    3. db.run

      sql挖空的话，最好采用如下这种方式：

    4. db.get

      大体类似于db.run，但是callback不太一样，这里callback的形式是function(err, row) {},

      且callback只对第一行结果生效

    5. db.all

      也是callback执行的条件不一样，是对所有结果生效

    6. db.each

      对每一行分别callback

    7. db.exec

      连续执行多条sql，出错就中断

    8. db.prepare

      返回一个Statement对象

    9. db.serialize

      串行模式，可以嵌套

    10. db.parallelize

      默认就是并行模式，这个是为了在串行模式中令某一部分并行使用的。

  4. Statement(instance是stmt)

    1. stmt.bind

      绑定参数

    2. stmt.reset

      重置已绑定的参数和游标

    3. stmt.finalize

      可以finalize某个Statement，之后所有的调用都会抛出错误。

      剩下的都类似上面了。

    4. stmt.run

    5. stmt.get

    6. stmt.all

    7. stmt.each
