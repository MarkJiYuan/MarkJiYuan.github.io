日志/2020-09-16

2020-09-16

上午：

  1. 文件里的文件的顺序是如何决定的，在文件夹的信息里面吗？以及如何操作系统如何排序的信息记录在哪里
  2. 注意python的全局变量，如果是不可变类型，要使用global关键字进行声明
  3. 写了directory_manager脚本，os.path.join很好用嘛
  4. 多用 dict.get ，默认返回值是None

中午：

  1. 算术平均 > 几何平均 > 调和平均

晚上：

  1. 原码，反码，补码

  2. 第一位是符号位，取1则表示负数。举例机器数是10010，真值是实际表达的数值，即-2。

  3. 原码就是符号位加上真值

  4. 补码是原码各位取反然后+1

  5. DRAM(Dynamic Random Access Memory) 动态随机存取存储器

  6. 取消别名 unalias

  7. pipenv 手动指定python interpreter path

  8. git remote -v 可以查看具体的地址（fetch和push）

  9. Flask

    1. 环境变量

    2. url_for 用来构建函数调用某函数的url (static节点是特殊的)

    3. render_template 渲染模版

    4. 使用app.test_request_context做测试

    5. make_response来构建一个响应对象。可以主动调用来改变一些headers。

    6. 使用jsonify而不是json.dumps，Content-type不同且jsonify没有空格

    7. redirect函数用于重定向

    8. abort用于提前退出请求

    9. 定制错误页面

    10. 消息闪现

    11. 日志（app.logger）

  10. 今天的真的第一次知道，太长见识了！Python的字符串之间居然不用加连接符！！
